!!! note "学习建议"
    1. 软件工程的平时项目考察的不只是最终得到的程序，更多在考察开发团队是否有按照标准化的团队开发流程去推进项目，考察团队的协作能力，需要通过一环接一环的开发文档与项目阶段展示进行呈现。

    2. 建议平时可按授课进度同步做题库的题以及 wzy 班 quiz，并记录错题以备期末复习。

    3. 期末时可将软工视为闭卷思政课，拼命去记。选择判断部分，原题必考，题库必刷。但是这部分只占 30 分，建议优先复习 50 分的画图题，弄懂 7 种图的画法，然后再做两道例题找找感觉。需要注意官方题库有错误，弄不懂就多问同学或急急急群友，可能是题目出错。简答题和画图题建议多做历年卷。很多画图题找不到答案，所以做了后可以多找同学的答案比对一下。简答题可以试试考前早读一下。
